## Featured Projects

### [Bible AI Assistant](https://github.com/t-timms/bible-ai-assistant)
Qwen3.5-4B fine-tuned with ORPO for biblical question-answering. Hybrid retrieval-augmented generation combining dense embeddings with keyword search. Constitutional AI self-critique guardrails for theological accuracy. Voice pipeline with speech-to-text and text-to-speech for hands-free interaction.

### [Godspeed](https://github.com/t-timms/godspeed-coding-agent)
Security-first open-source coding agent with a hand-rolled async ReAct loop. 4-tier deny-first permission engine, SHA-256 hash-chained audit trail, 200+ LLM providers via LiteLLM, 30+ built-in tools with JSON Schema validation, parallel + speculative tool dispatch, self-evolution via GEPA-style LLM-guided mutations, cost budget enforcement, multi-language verify gate, MCP server/client, and training data pipeline for fine-tuning tool-calling LLMs. Validated on SWE-bench Lite at 52.2% oracle best-of-5.

### [Sovereign Edge](https://github.com/t-timms/sovereign-edge)
Autonomous multi-agent personal intelligence system running on NVIDIA Jetson hardware. On-device inference with zero cloud dependencies. Privacy-preserving architecture for continuous operation at the edge.

### [Manna Trading](https://github.com/t-timms/manna-trading)
Multi-agent algorithmic trading system — 4-agent pipeline (Technical Analysis → Chief Strategist → Risk Manager → Execution Agent) with DeepSeek R1 reasoning at each stage. Kelly Criterion position sizing, Monte Carlo simulation for risk assessment, WebSocket streaming for real-time market data, and paper trading integration.

### [GPU Server Test Suite](https://github.com/t-timms/gpu-server-test-suite)
Comprehensive GPU diagnostic and validation toolkit modeled on NVIDIA DCGM architecture. Automated stress testing, memory validation, error correction detection, and health monitoring for GPU server fleets.

### [ML Lab](https://github.com/t-timms/ml-lab)
ML research control plane for systematic experiment management. Model registry with versioning, cloud training launcher with parallel sweep support, and automated metrics tracking across experiment runs.

### [ML Experiment Scaffold](https://github.com/t-timms/ml-experiment-scaffold)
Production-grade ML training infrastructure optimized for single-GPU homelabs. Config-driven pipeline with Unsloth fp8 quantization, torch.compile graph optimization, DeepSpeed ZeRO stages, lm-eval harness with vLLM serving, and multi-seed reporting for statistically sound results.

### [Manufacturing Quality Analytics](https://github.com/t-timms/manufacturing-quality-analytics)
SQL + Python ETL pipeline for semiconductor non-conformance report analysis. Supplier performance scoring with trend detection, defect Pareto distributions across manufacturing lines, yield rate dashboards, and automated alerting on quality threshold breaches.

### [LLM Wiki](https://github.com/t-timms/llm-wiki)
Git-backed knowledge wiki following Karpathy's LLM Wiki architecture. LangGraph-powered ingestion pipelines for structured and unstructured content. Version-controlled knowledge base with full diff history for LLM research and development reference.
